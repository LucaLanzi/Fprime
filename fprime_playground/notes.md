# Important Notes for working within Fprime

### Activating the virtual environment
Before working in Fprime you want to make sure you activate the virtual environment. This ensures that you work isolated from the global packages Python3 uses, where you can still use them, add more, or modify them and it will not affect your global python3 installation

### Run the following
`cd my_playground`

then

`. fprime-venv/bin/activate`

This navigates to your fprime project directory, and opens the virtual environment
